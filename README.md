# Student Management System

A simple web-based Student Directory application with full CRUD (Create, Read, Update, Delete) capabilities.

## 🚀 Features
- **Add Student**: Create new records with Name, Email, and Major.
- **View Directory**: Real-time display of all stored records.
- **Edit Records**: Update existing student information.
- **Delete Records**: Remove student data from the system.
- **Relational Database**: Powered by SQLite for efficient local storage.

## 🛠️ Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **Database**: SQLite3

## 📦 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/deadhearth01/Student-Management-System-0038-0044.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Student-Management-System-0038-0044
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and visit: `http://localhost:3001`
